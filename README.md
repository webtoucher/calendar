## Installation

The preferred way to install this extension is through [composer](http://getcomposer.org/download/).

Either run

```
$ php composer.phar require webtoucher/calendar "*"
```

or add

```
"webtoucher/calendar": "*"
```

to the ```require``` section of your `composer.json` file.
